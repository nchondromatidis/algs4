This class definitely works on the assumption that you have some kind of coding experience. But I will try to answer your questions. The variables can be given a value at different points throughout the code. It benefits programmers to use variables for programs because you can maintain functionality for different planned inputs. You could define N by passing it as an argument to a method or inside the method itself(p &q) in my example.

Quickfind(int N){ int p, q; }

So it would look like this inside main()

Quickfind(100){ int p, q; p = 1 .......... system.out.print(id[]) }

Public classes/methods/variables can be called or used by any other class Private classes/methods/variables can only be accessed by its parent class or friend classes This gets into OOP and is a more advanced concept.
